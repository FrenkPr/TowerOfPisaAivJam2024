# KitchenWall
"Kitchen Wall" game: a project realized on AIV Game Jam 2023.<br/>
The team was made of: 1 Programmer (me), 4 3D Graphic Designers and 1 Sound Designer.<br/>
The theme of the Jam was: "Beyond The Wall".
